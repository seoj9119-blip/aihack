# 역할 분담 제안

## 1. 프로젝트 매니저 / 기획 담당

- PRD 기반 전체 일정 관리
- 사용자 스토리 정리 및 우선순위 결정
- 팀 커뮤니케이션, 데모 준비, 리스크 관리

## 2. 프론트엔드 개발 담당

- React + Vite 앱 구조 구현
- 템플릿 입력 폼, 어조 선택 UI, 에디터 화면 개발
- 스타일링 및 반응형 레이아웃 완성

## 3. AI 통합 / 백엔드 담당

- OpenAI 등 외부 AI API 호출 로직 설계
- 프롬프트 통합, 템플릿 입력 -> 초안 생성 흐름 구현
- 학습 데이터 제외 옵션 등 보안/프라이버시 처리 검토

## 4. UI/UX 및 QA 담당

- 에디터 드래그 선택/부분 수정 UX 개선
- 사용자 흐름 테스트 및 버그 리포트
- 최종 결과물 복사/다운로드 사용성 점검
